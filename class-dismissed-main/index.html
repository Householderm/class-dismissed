Social Security Fairness Act — Fully Built In

WEP and GPO have been permanently repealed. Class Dismissed is fully updated — calculate your restored benefits, estimate retroactive back-pay, and maximize spousal and survivor SS strategies under the new law.

WEP Repealed

Your own SS benefit is no longer reduced because you receive a government pension. Full earned benefit restored.

GPO Repealed

Spousal and survivor SS benefits no longer offset by 2/3 of your pension. Previously zero benefit may now be substantial.

How to Use It
Choose how you want to run it

Three ways to use Class Dismissed — pick what's right for you.

🪟 Windows

1
Click Download Free above. The file ClassDismissed-RetirementPlanner.html saves to Downloads.
2
Double-click the file. Windows opens it in your default browser — Chrome, Edge, or Firefox all work.
3
Optional: Right-click → Send to → Desktop to create a shortcut for easy daily access.
4
No internet needed after download. Fully offline.

🍎 Mac

1
Click Download Free above. File saves to your Downloads folder.
2
Double-click to open in Safari, or right-click → Open With to choose Chrome or Firefox.
3
Optional: Drag the file to your Dock or Desktop for quick access.
4
No Gatekeeper warning (it's a web page). Fully private and offline.

Install on Windows

1
Download Class.Dismissed.Setup.1.0.0.exe using the Windows button above.
2
Double-click the installer. If SmartScreen appears, click "More info" → "Run anyway".
3
Choose install location, click Install. Desktop shortcut and Start Menu entry created automatically.
4
Launch from Desktop or Start Menu. Works fully offline.
SmartScreen Note: Because the app isn't commercially code-signed, Windows may show a warning on first run. Click "More info" → "Run anyway" to proceed safely.

Install on Mac

1
Download Class.Dismissed-1.0.0.dmg (Intel) or Class.Dismissed-1.0.0-arm64.dmg (Apple Silicon M1/M2/M3/M4) using the Mac buttons above.
2
Double-click the .dmg. Drag Class Dismissed into your Applications folder.
3
First launch: right-click the app → Open → click Open in the Gatekeeper dialog (one-time).
4
App is now in Applications. Works fully offline.
Gatekeeper Note: The app isn't notarized through Apple Developer Program. Right-click → Open the first time to bypass the warning. After that, it opens normally from your Dock.
